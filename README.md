# Sync Companion — Blank Flutter App

This workspace contains a minimal Flutter app scaffold suitable for mobile development.

Files added:
- `pubspec.yaml` — project metadata and dependencies
- `lib/main.dart` — minimal app entrypoint
- `.gitignore` — recommended ignores for Flutter projects

Quick steps to get started (PowerShell):

```powershell
# 1) Ensure Flutter is installed and on PATH
flutter --version

# 2) From this project root, fetch dependencies
flutter pub get

# 3) Run on a connected device or emulator
flutter run

# If you'd like a full platform directory structure, run:
flutter create .
```
